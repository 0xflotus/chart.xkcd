<summary>Install</summary>

#### Script Tag

```html
<div class="chart" style="width:600px; height: 300px;"></div>
<script src="path/to/chart.xkcd.js"></script>
<script>
  const div = document.querySelector('chart');
  chartXKCD(div, {...})
</script>
```

#### npm

```bash
npm i chart.xkcd
```
